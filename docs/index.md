# 奇跡の一枚 – サポート

ようこそ。こちらは iOS アプリ「奇跡の一枚」のサポートページです。

## よくある質問
- アプリが開けない/クラッシュする → 端末の再起動、アプリの再インストールをお試しください。
- 写真にアクセスできない → 設定 > プライバシー > 写真 から本アプリのアクセスを許可してください。

## 動作環境
- iOS 16 以降（最新の iOS でのご利用を推奨）
